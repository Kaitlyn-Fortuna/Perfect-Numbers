# Perfect-Numbers

By: Kaitlyn Fortuna

Language: Python

A perfect number is a number where the sum of its divisors is equal to the number itself. This python code will take input two integers a starting value and an ending value and check for perfect numbers in that range. The perfect numbers that you find in the range should be printed out as a list of comma-separated items. If no perfect numbers are found in the range nothing should be printed out.
